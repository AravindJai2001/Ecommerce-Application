# 🛒 Easy Shopping Platform

Easy Shopping is a full-stack e-commerce application built using modern web technologies.

---

## 🚀 Tech Stack

### Frontend
- React
- Axios
- Bootstrap / Tailwind

### Backend
- Spring Boot
- Spring Data JPA
- Hibernate

### Database
- PostgreSQL

---

## 📂 Project Structure

easy-shopping-platform/
│
├── easy-shopping-ui
├── easy-shopping-api

---

## ⚙️ How To Run Locally

### Backend
cd easy-shopping-api
mvnw.cmd spring-boot:run

Runs on: http://localhost:8080

### Frontend
cd easy-shopping-ui
npm install
npm start

Runs on: http://localhost:3000

---

## ✨ Features

- User Registration & Login
- Product Listing
- Add to Cart
- Order Management
- Role Based Access (Admin/User)
